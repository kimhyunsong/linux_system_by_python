# linux_system_by_python

# 시작하기 
```
git clone https://github.com/kimhyunsong/linux_system_by_python.git
# filebrowser 설치
curl -fsSL https://raw.githubusercontent.com/filebrowser/get/master/get.sh | bash
# python3-prctl 설치
sudo apt-get install -y python3-prctl
# 실행
python3 main.py
```
